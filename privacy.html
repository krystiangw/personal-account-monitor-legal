Kanga P2P Clicker

Privacy Policy

Effective date: 8 May 2026

Overview

Kanga P2P Clicker is a private tool used by its operator for personal account monitoring and payment matching. It is not a public service and is not offered to other users.

Data Controller

The data controller is the private individual operating this tool. For data protection matters, use the contact email provided in the related Enable Banking application registration.

Data Processed

When the operator authorizes account access through an Open Banking provider, the tool may read account transaction data needed to identify incoming PLN payments. This can include transaction amount, currency, booking date, counterparty name, payment reference and transaction identifier.

The tool may also process Telegram message metadata and message content from the operator's private chat with the Kanga P2P bot, limited to the data needed to match P2P orders and payment confirmations.

Purpose

Data is processed only to match incoming bank payments with the operator's own P2P transaction notifications and to help confirm when funds have been received.

Legal Basis

Processing is based on the operator's own consent and request to access their account information through the Open Banking provider.

Sharing

The tool does not sell personal data and does not share data with third parties for advertising or analytics. Account access is provided through the selected Open Banking provider and the operator's bank or financial institution as part of the authorization flow.

Storage

The tool is designed to run locally for the operator. Operational logs may be stored locally for debugging and audit purposes. Logs should not contain bank login credentials and should be deleted when no longer needed.

Security

The tool uses read-only account information access. It does not request permission to initiate payments. Open Banking credentials, private keys and Telegram session data must be kept secret by the operator.

Retention

Local logs and configuration are retained only for as long as required for operating and troubleshooting the tool. The operator may delete local logs at any time.

Your Rights

Because this is a private tool used only by its operator, the primary data subject is the operator. Any data protection request can be sent to the contact email provided in the Enable Banking application registration.
